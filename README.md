# 달과 바다 — 웹 플립북 전자책 데모

외부 라이브러리 없이 순수 HTML/CSS/JS로 만든 책장 넘김(flipbook) 방식 웹 전자책 뷰어 샘플입니다.

## 기능

- CSS 3D 변환 기반 양면 페이지 플립 (책등 음영·종이 두께 표현)
- 페이지 이동: 좌우 화살표, 키보드(←/→, Home/End), 스와이프, 하단 슬라이더
- 썸네일 드로어로 페이지 바로 가기
- 확대/축소 및 드래그 이동 (Ctrl + 휠 줌 지원)
- 전체화면 모드
- Web Audio로 합성한 종이 넘김 효과음 (켜고 끄기 가능)

## 실행

`index.html`을 브라우저에서 열면 바로 작동합니다. 빌드나 서버가 필요 없습니다.

## 콘텐츠 교체

`index.html` 안의 `pages` 배열에 페이지 HTML을 넣는 구조입니다.
이미지 기반 전자책으로 쓰려면 각 항목을 `<img src="pages/1.jpg">` 형태로 바꾸면 됩니다.

## 라이선스

자유롭게 수정·사용 가능한 데모 코드입니다.
